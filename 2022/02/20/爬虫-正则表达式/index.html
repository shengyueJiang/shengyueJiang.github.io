正则表达式

python的re库提供了整个正则表达式的实现，利用这个库，可以在python中使用正则表达式。

元字符

元字符：具有固定含义的特殊符号

常用元字符（每一个元字符默认只匹配一个字符）：

  • . ：匹配除了换行符以外的字符
  • \w : 匹配字母，数字，下划线
  • \s : 匹配任意的空白符
  • \d : 匹配任意数字
  • \n : 匹配换行符
  • \t : 匹配制表符
  • ^ : 匹配字符串的开始
  • $ : 匹配字符串的结尾
  • \W : 匹配非字母，数字，下划线
  • \S : 匹配非任意的空白符
  • \D : 匹配非任意数字
  • a|b : 匹配字符a或者字符b
  • ( ) : 匹配括号内的表达式，也表示一个组
  • [ ] ：匹配字符组中的字符
  • [ ^] ：匹配除了字符组中的字符

量词

  • * : 可以重复出现零次或者很多次
  • + : 可以重复出现一次或者很多次
  • ？ : 可以重复出现零次或者一次
  • {n} : 重复n次
  • {n,} : 重复n次或更多次
  • {n,m} : 重复n次到m次

贪婪匹配和惰性匹配

  • .* ：贪婪匹配
  • .*？ ：惰性匹配

.*表示会匹配尽可能多的字符.*?表示尽可能的少匹配字符。
